# Security Policy for DirWatchBot

## Reporting a Vulnerability
If you discover a security issue, please **do not** open a public issue. Instead, email us directly at `topherbot@proton.me` with a clear description of the problem, steps to reproduce, and potential impact.

## Supported Versions
Only the latest commit on the `main` branch receives security updates.

## Responsible Disclosure
We commit to responding within 48 hours of receiving a report and will provide a fix or mitigation as soon as reasonably possible.
